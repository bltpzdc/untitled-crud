# Первый прототип

Позволяет пользователям:
- отправить на сервер вывод инструмента и метаданные
    - формат отправки архив, в котором на верхнем уровне лежат:
        - папка `crashes` с содержимым только этого запуска
        - метаданные `metadata.json` с именем отправителя и временем запуска инструмента
        - файл конфигурации `configuration.toml`
        - лог анализатора `diffuzzer.log`
    - по результатам отправки получает id в системе
- посмотреть уже загруженные на сервер отчеты по их метаданным
- скачать ранее загруженный вывод инструмента по его id в системе
